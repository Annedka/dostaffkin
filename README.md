# Dostaffkin

Приложение для оформления и отслеживания заказов доставки.

## Основные возможности
- Оформление заказа
- Отслеживание статуса доставки
- Просмотр истории заказов
- Адаптивный интерфейс

## Технологии
- Язык: TypeScript
- Фреймворк: Angular
- Стили: CSS
- Сборка: Angular CLI

## Структура
- `src/app/pages/order` — оформление заказа
- `src/app/pages/track` — отслеживание заказа
- `src/app/services/delivery-api.ts` — работа с API доставки
- `src/app/header` — шапка приложения

## Установка и запуск
```bash
npm install
npm start
```
